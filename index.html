Responsibility Activities Member Competencies
 Responsibility
  • Bertanggung jawab atas efektivitas fungsi pengelolaan internal use case management, external use case management, data analytics consulting, dan pemberdayaan pada chapter data scientists
  • Melaksanakan implementasi aktivitas-aktivitas budaya organisasi
  • Membangun relasi dengan unit kerja lain dan key person (eksternal/ internal) untuk mendukung penyelesaian pekerjaan
  • Memastikan kompetensi yang dipersyaratkan bagi pekerjaan ditingkatkan, dengan mempelajari keahlian /pengetahuan yang sesuai untuk menyelesaikan pekerjaan dengan efektif dan menciptakan inovasi kerja
 Activities
  • Melaksanakan pengelolaan fungsi resource management untuk memenuhi kebutuhan data scientists di tribe atau unit lain
  • Mengembangkan text, voice, dan video analytics untuk kepentingan pengelolaan insight atas data yang berasal dari: social media , SMS, email , network , agent atau customer service note dan bentuk lainnya untuk dilakukan analisis
  • Membangun dictionary dan training set untuk melakukan classification
  • Memberikan insight atas pengolahan data yang dilakukan untuk improvement
  • Melakukan business process understanding , data understanding , dan data preparation dalam membuat data model
  • Menampilkan data insight untuk ditampilkan ke front line berupa targeted caring , dan targeted offering dari hasil data analytics
  • Membangun predictive model untuk improvement program internal maupun eksternal, misalnya meliputi: churn model, collection model , network analytics model untuk menjadi program proaktif dalam rangka menurunkan keluhan pelanggan
  • Membangun competency dan analytics engine untuk text analytics, leads generation, predictive model dan suggested caring/ offering
  • Mengembangkan internal & external use case management yang menjadi library use case untuk mendukung kebutuhan tribe
